# ↪︎ Como jogar

Use as **setas do teclado** para controlar a cobra:

| Tecla | Ação |
|------|------|
⬆️ | mover para cima |
⬇️ | mover para baixo |
⬅️ | mover para esquerda |
➡️ | mover para direita |

↪︎ Objetivo:
- Comer a comida
- Crescer o tamanho da cobra
- Evitar bater nas paredes ou em si mesma
